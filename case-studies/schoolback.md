# SchoolBack - Caso de estudio privado

## Resumen

Sistema escolar privado para administrar usuarios, registros, permisos y modulos operativos. Se documenta como caso de estudio porque el codigo real y la informacion del sistema no deben publicarse.

## Mi rol

Desarrollador Backend / Full Stack Jr.

## Tecnologias

- Node.js
- Express
- Prisma
- MySQL
- React
- Autenticacion y permisos

## Responsabilidades

- Apoyo en endpoints REST para modulos administrativos.
- Consulta y organizacion de informacion por usuario o modulo.
- Validaciones de formularios y reglas de negocio.
- Manejo de estructura de datos con enfoque en permisos.
- Pruebas funcionales y ajustes para uso administrativo.

## Privacidad

No se publican nombres de alumnos, usuarios reales, credenciales, URLs internas, capturas productivas ni codigo fuente privado.

## Version demo recomendada

Crear una demo separada con:

- Usuarios ficticios.
- Roles genericos.
- Registros simulados.
- Pantallas recreadas sin datos reales.
