# Delice Flow - Caso de estudio privado

## Resumen

Plataforma administrativa para restaurante enfocada en menu, pedidos, inventario, proveedores y movimientos operativos.

## Mi rol

Backend / Full Stack Jr.

## Tecnologias

- Node.js
- Express
- Prisma
- MySQL
- React

## Responsabilidades

- Diseno de entidades para menu, pedidos, inventario y proveedores.
- Desarrollo de endpoints REST.
- Uso de Prisma para acceso a datos.
- Integracion con frontend administrativo.
- Pruebas funcionales de modulos.

## Privacidad

La version real se mantiene privada. No se publican ventas, proveedores, movimientos reales, credenciales ni codigo original.

## Version demo recomendada

Crear `restaurant-admin-demo` con:

- Productos ficticios.
- Proveedores genericos.
- Pedidos simulados.
- Inventario recreado.
