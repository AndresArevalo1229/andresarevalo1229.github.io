# inventory-wms-demo

Demo publica para explicar logica de inventarios tipo WMS con datos ficticios.

## Objetivo

Mostrar control de existencias, lotes, ubicaciones y movimientos sin publicar informacion privada.

## Stack sugerido

- React
- Node.js
- MySQL
- TypeScript

## Modulos

- Productos genericos
- Lotes
- Ubicaciones
- Entradas y salidas
- Reportes recreados
